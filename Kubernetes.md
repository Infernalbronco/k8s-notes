# Kubernetes
1. It is container orchestration tool.
2. Helps you manage containerized applications.
3. Created by google.
4. Containers are used in microservices.

##### Features of container orchestration tool
1.  High availability or no downtime.
2.  Scalability.
3.  Disaster recovery.
